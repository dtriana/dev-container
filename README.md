# VS Code Remote Containers for Logic Apps local development

A great option for Logic Apps local development is VS Code Remote Containers, which will open the project in your local VS Code using the [Dev Containers extension](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-containers)

You need Docker installer for this to work.
